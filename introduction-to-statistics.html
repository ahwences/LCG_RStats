4 Introduction to Statistics

4.1 Big Picture

4.1.1 Data

  • Random variables live in the theoretical (mathematical) world
  • We assume that the variability of the data is adequately consistent with the variability that would occur in a random sample from a given distribution/random process.
    • Data itself is assumed to be random variables so that we can focus on the best way to reason from random variables to inferences about (theoretical) parameters.
  • Beware, often the language of statistics takes a convenient shortcut by blurring the distinction between data and random variables.

4.1.2 Models

  • Incorporate theoretical assumptions and subjunctive statements (of the form if â¦ then â¦).
  • Inference is based on what would happen if the data were to be random variables distributed according to the statistical model.
  • The modelling assumption would be reasonable if the model were to describe accurately the variation in the data.

4.1.3 Probability

  • Aleatoric Probability: The use of probability to describe variation. E.g. âThe probability of rolling a 3 in a fair dice is 1/6â.
  • Epistemic Probability: The use of probability to express knowledge (or beliefs). E.g. âIâm 90% sure that the capital of MÃ©xico is CDMXâ.

4.2 Likelihood Function

4.2.1 Definition

Definition 4.1 Let \(\{\mathcal{P}_\theta\}_{\theta\in\Theta}\) be a set of probability models (e.g.Â distributions) indexed by a (possibly a vector) parameter \(\theta\) that lives in some space \(\Theta\) (e.g.Â \(\mathbb{R}^n\)).

  • Assume that, for each \(\theta\), if \(\bar{X}=(X_1,\dots,X_n)\) is a random vector whose distribution is given by \(\mathcal{P}_{\theta}\), then \(\bar{X}\) has density \(f(x\vert \theta)\).

  • Consider a fixed set of values \(\bar{x}=(x_1,\dots,x_n)\).

Then the likelihood function \(L(\bar{x}\vert \theta)\) is the function of \(\theta\) given by \[L(\bar{x} \vert \theta) = f(\bar{x}\vert \theta).\]

Remark. Note that if, instead we regard \[ L(\bar{x} \vert \theta) = f(\bar{x}\vert \theta) \] as a function of \(\bar{x}\) (with \(\theta\) fixed) then what we have is simply the density function of \(\bar{x}\) under the model \(\mathcal{P}_\theta\).

\

Example 4.1 Consider the family of probability models for a random vector \(\bar{X}\) of size \(n\) whose entries are all independent and identically distributed \(Gamma(\alpha,\beta)\). Then for any vector \(\bar{x}=(x_1,\dots,x_n)\), and using the independence assumption for the first equality, we have \[ \begin{aligned} L(\bar{x}\vert \alpha,\beta) &= \prod_{k=1}^n L(x_i\vert \alpha,\beta) \\ &= \left(\frac{\beta^\alpha}{\Gamma(\alpha)}\right)^n \prod_{k=1}^n x_i^{\alpha-1}e^{-\beta x_i}\\ &= \left(\frac{\beta^\alpha}{\Gamma(\alpha)}\right)^n \left(\prod_{k=1}^n x_i\right)^{\alpha-1} e^{-\beta\sum_{k=1}^n x_i}. \\ \end{aligned}. \] If we now change the family of models \(\{\mathcal{P_\theta}\}_{\theta\in\Theta}\) and assume that the entries of \(\bar{X}\) are still independent but not identically distributed, for example if we assume that \(X_i\) has distribution \(Gamma(\alpha_i, \beta)\) (they share the parameter \(\beta\) but not necessarily the parameter \(\alpha\)), then we obtain \[ \begin{aligned} L(\bar{x}\vert \alpha,\beta) &= \prod_{k=1}^n L(x_i\vert \alpha,\beta) \\ &= \frac{\beta^{\sum_{k=1}^n \alpha}}{\prod_{k=1}^n\Gamma(\alpha_i)} \left(\prod_{k=1}^n x_i^{\alpha_i-1}\right) e^{-\beta \sum_{k=1}^n x_i} \\ \end{aligned}. \]

4.2.2 Maximum Likelihood Estimator MLE (Point Estimate)

The likelihood functions can be used to obtain a point estimate that typically has very desirable properties.

Definition 4.2 Let \(\{\mathcal{P}_\theta\}_{\theta\in\Theta}\) be a family of probabilistic models. Let \(\bar{x}=(x_1,\dots,x_n)\) be a sample (a set of values). The maximum likelihood estimator for \(\theta\) based on the sample \(\bar{x}\) is given by \[ \hat{\theta} = arg\max_{\theta\in\Theta} L(\bar{x}\vert \theta), \] it can be interpreted as the paramter \(\hat{\theta}\) under which the data \(\bar{x}\) is better explained by \(\mathcal{P}_{\theta}\), where the criterion to say that a model \(\mathcal{P}_{\theta}\) âexplains the data betterâ is simply that of saying that it is more likely to observe the data \(\bar{x}\) under the model \(\mathcal{P}_\theta\).

Example 4.2 Let \(\bar{X}=(X_1,\dots,X_n)\) be i.i.d \(normal(\mu,1)\). Then

\[ L(\bar{X}\vert \theta) = \frac{1}{(2\pi)^{n/2}} e^{-\frac{1}{2}\sum_{k=1}^n(x_k-\mu)^2}. \] The equation \[ \frac{d}{d\mu}L(\bar{X} \vert \theta) = 0 \] reduces to \[ \sum_{k=1}^n(x_k-\mu) = 0 \] which has solution \[ \hat{\mu} = \frac{1}{n} \sum_{k=1}^N X_k = \tilde{X}. \]

4.3 Hypothesis testing

Definition 4.3 A hypothesis is a statement about a population/model parameter.

The goal of a hypothesis test is to decide, based on a sample from the population, which of two complementary hypothesis is true.

Definition 4.4 The two complementary hypothesis in a hypothesis testing problem are called the null Hypothesis (\(H_0\)) and the alternative hypothesis (\(H_1\)).

\

Definition 4.5 A hypothesis testing procedure as rule that sepecifies:

  1. For which sample values the decision is made to accept \(H_0\) as true.
  2. For which sample values \(H_0\) is rejected (in ``acceptanceââ of \(H_1\)).
The region of the sample space on which \(H_0\) is rejected is called the rejection region, while the region in which \(H_0\) is accepted is called the acceptance region.
